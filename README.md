## Запуск приложения

Установка зависимостей

```sh
npm install
```

Локальный запуск проекта

```sh
npm run start
```

## Environment

Шаблон переменных находится в файле .env.sample
Создайте файл .env в корне проекта и скопируйте туда переменные.
